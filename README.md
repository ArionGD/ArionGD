### 📁 Featured Project: ORBIT 🪐

> **A Secure, Self-Hosted Document Compliance & Expiration Manager**

ORBIT tracks certificates, licenses, and permits with high visual fidelity and zero database bloat:
*   🌌 **Concentric Integrations Slide:** Autoplay integration slider with planetary orbit animations.
*   📂 **Cloud-First Sync:** Integrates with **Google Drive API** via OAuth 2.0 to upload payloads directly, falling back gracefully to local media folders when credentials aren't present.
*   📨 **SMTP Expiration Sweeper:** Custom Django scheduler command sweeps the database daily to mail grouped urgency warnings.
*   🛡️ **Multi-Role Manager Console:** Provides role-based permissions (`user` and `manager`) allowing managers to audit user vaults, toggle account status flags via **HTMX**, and preview mail templates dynamically.
